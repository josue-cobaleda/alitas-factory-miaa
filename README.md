# Alitas Factory - Visión Artificial para Analítica de Clientes

## Descripción
Proyecto de Maestría en Inteligencia Artificial (ICESI) que implementa modelos de visión artificial (YOLOv11 y Transformer) para:
- Detección y conteo automático de clientes
- Análisis de ocupación en tiempo real
- Generación de métricas de comportamiento
